# Mastermind
### Guillem Soto
## Robust
El codi es pot executar sense problema, a no ser que algú no posi un int a l'hora d'escollir número.
## Claredat i Ordre
El que es busca fer està clar en tot moment gràcies als comentaris.
## Variables i constants
Les variables tenen un nom clar i la gran majoria dels textos queden com a constant.
## Gestió d'errors
Es gestionen els errors dins del que un usuari que no vagi a buscar-los expresament podria experimentar.
## Jocs de Proves
Hi ha jocs de proves (imatges al repositori)
## UX/UI
L'interfaç s'entén perfectament a l'hora de jugar.
